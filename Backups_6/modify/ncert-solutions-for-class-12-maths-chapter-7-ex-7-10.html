प्रश्न 10.
यदि f(x) = \(\int_{0}^{x} t \sin t d t\) है, तब f'(x) है-
(A) cos x + x sin x
(B) x sin x
(C) x cos x
(D) sin x + x cos x
हल-
f(x) = \(\int_{0}^{x} t \sin t d t\)

अतः f'(x) = -[cos x – x sin x] + cos x
= – cos x + x sin x + cos x
= x sin x
अतः विकल्प (B) सही है।
