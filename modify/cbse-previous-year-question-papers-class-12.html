7 Steps to Consider for CBSE Class 12 Previous Year Papers

  • Go through the complete question paper.
  • Create a strategic plan so as to attempt the question papers methodically.
  • Make questions Attempt plan – decide which question to attempt first and which to leave.
  • If you don’t know an answer, just skip the question. Attempt questions only when 90-100% sure.
  • Once done with papers, check the amount of time taken.
  • Start moderating your answers honestly.
  • Go through each and incorrect answers so as to not repeat the same mistakes in the exam.
