Question 4.
The domain of f(x) = sin -1 (-x²)is
(a) (-1, 1)
(b) [-1, 1]
(c) (-1, 1]
(d) [-1, 1)
Solution:
(b) We know that domain of sin -1 x is [-1, 1]
Therefore, f(x) = sin -1 (-x²) is defined for all x satisfying
-1 ≤ -x² ≤ 1
⇒ 1 ≥ x² ≥ -1
⇒ 0 ≤ x² ≤ 1
⇒ x² ≤ 1 ⇒ x² – 1 ≤ 0
⇒ (x – 1)(x + 1) ≤ 0
⇒ -1 ≤ x ≤ 1
Hence, the domain of f(x) is [-1, 1].

Question 31.
Consider the function f : R + → [4, ∞) defined by f(x) = x² +4, where R + is the set of all non-negative real numbers. Show that f is bijective.
Solution:
We have a mapping f : R + → [4, ∞) given by
f(x) = x² + 4.
To prove f is invertible.
For f to be one-one.
Let x 1 , x 2 ∈ R + be any arbitrary elements, such that
f(x 1 ) = f(x 2 )
⇒ x² 1 + 4 = x² 2 + 4 ⇒ x² 1 – x² 2 = 0
⇒ (x 1 – x 2 ) (x 1 + x 2 ) = 0
⇒ x 1 – x 2 = 0 [∵ x 1 + x 2 ≠ 0 as x 1 , x 2 ∈ R + ]
⇒ x 1 = x 2
So, f is one-one.
For f to be onto
Let y ∈ [4, ∞) be any arbitrary element and let y = f (x)


such that f(x) = y
So, f is onto.
Hence, f is bijective. Hence proved.
